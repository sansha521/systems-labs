# systems-labs
This is a repository of the labs that were done for the course Intro to Operating Systems.

## Labs

- **[CPU Lab](cpu/)**: Implementation and analysis of CPU scheduling algorithms.
- **[Virtual Memory Lab](virtual/)**: Exploration of virtual memory systems, paging, and address translation.
- **[Memory Lab](mem/)**: Memory management techniques, including allocation and deallocation strategies.

## Getting Started

Each lab directory contains its own README with specific instructions, code, and documentation. Clone this repository and navigate to the desired lab directory to begin.

## Requirements

- C compiler (e.g., GCC)
