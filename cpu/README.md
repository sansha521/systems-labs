# CPU Lab

This lab focuses on CPU scheduling algorithms, a fundamental concept in operating systems. You'll implement and analyze different scheduling policies to understand how they affect process execution and system performance.

## Objectives

- Understand various CPU scheduling algorithms (e.g., FCFS, SJF, Round Robin, Priority Scheduling)
- Implement scheduling logic in C
- Analyze algorithm performance metrics (waiting time, turnaround time, CPU utilization)
- Compare different scheduling strategies

## Files

- `cpu.c` - Main implementation file
- `oslabs.h` 

## Compilation and Execution

```bash
gcc cpu.c -o cpu
