# Memory Lab

This lab covers memory management techniques, including dynamic memory allocation, deallocation, and fragmentation analysis.

## Objectives

- Understand memory allocation strategies (e.g., first-fit, best-fit, worst-fit)
- Implement custom malloc/free functions
- Analyze memory fragmentation and efficiency
- Handle memory leaks and corruption

## Files

- `mem.c` - Main implementation file
- Other supporting files as needed

## Compilation and Execution

```bash
gcc mem.c -o mem
./mem
