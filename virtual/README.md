# Virtual Memory Lab

This lab explores virtual memory concepts, including paging, address translation, and memory management techniques used in modern operating systems.

## Objectives

- Understand virtual memory architecture
- Implement page table management
- Handle address translation and page faults
- Analyze memory access patterns and performance

## Files

- `virtual.c` - Main implementation file
- `oslabs.h`
